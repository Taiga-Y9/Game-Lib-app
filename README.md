# 🎮 GameLib — マイゲームライブラリ

> ゲームコレクションを管理・記録するパーソナルゲームライブラリアプリ

**公開URL:** https://game-lib-app.vercel.app/

---

## 概要

**GameLib** は、プレイしたゲーム・積んでいるゲームを一元管理するための個人向けWebアプリです。

ゲームごとに「積みゲー / プレイ中 / クリア済み / 100%達成」のステータスを付け、プレイ時間・評価・感想メモを記録できます。さらに「プレイ日記」機能でプレイ中の出来事をタイムライン形式で残せます。

### 開発の背景・経緯

ゲームを買っても積むことが多く、「どのゲームをどこまでやったか」が分からなくなる問題を解決したくて開発しました。既存サービスはPCゲームに偏っていたり、UIが重かったりと自分の用途に合わないものが多く、シンプルで使いやすい自作ツールを作ることにしました。

## 特徴と機能の説明

### 🗂 ステータス管理 × フィルタリング

ゲームに以下4つのステータスを設定できます：

| ステータス    | 意味                     |
| ------------- | ------------------------ |
| 📦 積みゲー   | 購入・入手済みだが未着手 |
| 🎮 プレイ中   | 現在進行中               |
| ✅ クリア済み | エンディングまで到達     |
| 🏆 100%達成   | 全実績・全コンプリート   |

トップページのタブで各ステータスを瞬時に切り替え、タイトル検索・カテゴリフィルターと組み合わせて目的のゲームをすぐに見つけられます。

### 統計ダッシュボード & 積みゲー警告

総ゲーム数・プレイ中・積みゲー数・総プレイ時間（時間単位）をトップに常時表示します。積みゲーが **5本以上** になると赤いアラートバナーでお知らせし、クリア率も表示します。購買衝動を抑える心理的ストッパーとして機能します。

<img width="938" height="130" alt="Image" src="https://github.com/user-attachments/assets/93d5309a-fd40-4f00-bb55-46d1269e4bd6" />

### プレイ日記

ゲーム詳細ページから短い日記を投稿できます。「ラスボス倒した！」「今日はここまで」といった一言メモをタイムライン形式で蓄積。ゲームのリプレイ時や記憶の振り返りに使えます。投稿・閲覧は認証済みユーザーのみ可能です。

### プレイ時間 & ★評価

プレイ時間を時間・分単位で記録し、カードにも詳細ページにも表示されます。星5段階の評価も登録でき、ライブラリ全体でどれだけ時間を投資したか可視化できます。

### 管理者認証

Supabase Auth のメール・パスワード認証で管理者のみ投稿・編集・削除が可能です。未認証ユーザーはライブラリの閲覧のみに制限されており、`useRouteGuard` フックで管理ページへのアクセスを保護しています。

### ダークテーマ UI

Tailwind CSS v4 を使い、スレートとパープル・インディゴのグラデーションで統一したダークテーマを採用。ゲームの世界観に合う没入感のあるデザインにこだわりました。

## 工夫した点・苦労した点

### Prisma v7 のグローバルシングルトンパターン

Next.js の開発環境では Hot Reload のたびに Prisma Client が新規作成されコネクションが枯渇する問題があります。`global` オブジェクトにインスタンスをキャッシュするシングルトンパターンを実装し、開発・本番どちらでも安定した接続を確保しました。

### Supabase Auth + ルートガードの実装

`useAuth` フックでセッション状態をリアルタイムに監視し、`useRouteGuard` で未認証時に自動リダイレクトする仕組みを実装しました。管理画面全体を `AdminLayout` でラップすることで、認証チェックを一元化しています。
